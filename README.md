# Frog WinForms App

Небольшая игра на C# с использованием WinForms, где игрок управляет лягушкой и должен переправить лягушек на противоположную сторону за минимальное колличество ходов.

Сделал этот проект, чтобы потренироваться в:
- работе с WinForms
- игровой логике
- обработке ввода с клавиатуры
- работа с картинками и перемещением их

---

## 🎯 Что это
аркадная игра: кликом управляешь лягушкой, перемещая на свободную кувшинку, можно перепрыгивать через одну.
Условие победы: пройти уровень за 24 хода. Если ходов больше — условное "поражение".

## 🎮 Что реализовано

- Управление с клавиатуры
- Движение лягушки при клике на неё
- Клик по лягушке — попытка переместить её на пустую клетку
- Проверка расстояния: лягушка может прыгнуть, если расстояние до пустой клетки ≤ 2 клетки
- Обновление состояния игры
- Счётчик ходов: увеличивается при каждом успешном прыжке
- Условие победы: все правые лягушки находятся правее центральной пустой клетки
- Условие поражения: игра пройдена, но потрачено более 24 ходов
