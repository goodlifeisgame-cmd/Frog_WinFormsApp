# Frog WinFormsApp

## Описание
FrogWinFormsApp – это логическая головоломка, вдохновлённая классической игрой "Лягушки-перепрыгушки" (или "пятнашки" с перестановкой). 
Вам нужно поменять местами две группы лягушек (левых и правых), используя одну пустую клетку. Лягушка может прыгать на пустое место, если расстояние не превышает две клетки.
Игра считается пройденной, когда все правые лягушки оказались на правой стороне от центрального пустого поля.

### Условие победы: пройти уровень за 24 хода. Если ходов больше — условное "поражение".

## Техническая часть

### Стек технологий
- **Тип приложения:** Десктопная игра для Windows
- **Язык:** C# на .NET 10
- **UI-фреймворк:** Windows Forms

 ### Архитектура
Монолитное приложение с логикой, размещённой непосредственно в коде формы `GameMainForm`.

### Основные механики
- Клик по лягушке — попытка переместить её на пустую клетку
- Проверка расстояния: лягушка может прыгнуть, если расстояние до пустой клетки ≤ 2 клетки
- Счётчик ходов: увеличивается при каждом успешном прыжке
- Условие победы: все правые лягушки находятся правее центральной пустой клетки
- Условие поражения: игра пройдена, но потрачено более 24 ходов

### Управление
- **Мышь** (клик по картинке лягушки)
- **Клавиатура** (через `ToolStripMenuItem` — рестарт)

### Особенности реализации
- Игровое поле фиксировано: 8 лягушек (4 слева, 4 справа) + 1 пустая клетка в центре
- Проверка расстояния через вычисление разницы координат `Location.X` и деление на ширину клетки
- После победы/поражения показывается полноэкранная картинка (`EndGamePictureBox.Size = this.Size`)
- Рестарт сбрасывает счётчик ходов и возвращает всех лягушек на стартовые позиции
